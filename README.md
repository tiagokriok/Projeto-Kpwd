# Projeto-Kpwd
Projeto simples que possui um fácil entendimento. Essa projeto tem o intuito o meu aprendizado e passar o conhecimento que adquiri para quem tenha interesse.
# Funcionamento
O arquivo kpwd.py será executado via terminal, onde será feito uma pergunta de qual o tamanho de senha você deseja e a retornará. Já o arquivo telakpwd.py quando executado será aberto uma GUI, nessa tela possui a pergunta de quantos caracteres deseja sua senha, a senha será retornada no campo de texto abaixo.
